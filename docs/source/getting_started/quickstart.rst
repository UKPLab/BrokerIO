Quickstart Guide
=================

This guide will help you get started with the NLP Broker.

You can either install the NLP Broker on your local machine or use the NLP Broker as a service.
For a local installation, please follow the instructions in the :doc:`installation <./installation>` section.
If you want to use the NLP Broker as a service, you will find the API under:

- http://peer.ukp.informatik.tu-darmstadt.de:4852 (for main)
- http://peer.ukp.informatik.tu-darmstadt.de:4853 (for dev)

Usage
-----

The NLP Broker is a websocket service.
